# Hotel Management System

A full-stack Java web application developed to manage hotel reservations efficiently using JSP, Servlets, JDBC, and MySQL.

##  Project Overview
The Hotel Management System is a CRUD-based web application that allows hotel staff to manage customer reservations through a simple and user-friendly interface. The system follows the MVC architecture and demonstrates backend integration using Java Servlets and JDBC.

##  Features
- Add new hotel reservations
- View all reservations
- Update existing reservation details
- Delete reservations
- Responsive and clean UI using HTML and CSS

##  Technologies Used
- **Frontend:** HTML, CSS, JSP  
- **Backend:** Java, Servlets, JDBC  
- **Database:** MySQL  
- **Server:** Apache Tomcat 10  
- **IDE:** Eclipse  
- **Version Control:** Git & GitHub  

##  Project Architecture
- **Model:** Java classes & DAO layer for database interaction  
- **View:** JSP pages for UI  
- **Controller:** Servlets handling business logic and routing
